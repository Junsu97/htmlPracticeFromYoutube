input 태그

사용자로부터 값을 입력받을 수 있는 대화형 컨트롤(또는 '필드')을 나타낸다.
기본적으로 인라인 요소이며, 단일태그이다.

input의 핵심, type 속성

type의 값에 따라 입력 요소의 형태나 입력 데이터 유형등이 달라진다.
사용 가능한 type은 20여 가지이며, 기본값은 text이다.

위 요소는 전부 input태그로 만든 요소이며, 클릭도 입력 유형 중 하나이다.


input 태그에는 name 식별자를 추가할 수 있다.
이는 각각의 입력 항목에 대한 이름이다.

name 식별자를 통해 input을 식별하는 것은 나중에 서버 프로그래밍이나 javascript에서 원활한 처리를 위해 꼭 필요한 작업이다.

실습

placeholder 속성은 메시지가 입력되기 전 나타나는 문자열이며, maxlength는 text의 최대 길이이다.



value 속성을 토해 버튼에 글자를 나타낼 수 있다.
